# 🌐 Language Translator

## 📌 Description
This project is a Language Translator built using HTML, CSS, and JavaScript. It translates text between different languages using an API.

## 🚀 Features
- Enter text
- Select source & target languages
- Translate text instantly
- Copy translated text
- Text-to-speech (voice output)
- Speech-to-text (mic input)

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- Translation API

## ▶️ How to Run
Download the project and open index.html in your browser.
